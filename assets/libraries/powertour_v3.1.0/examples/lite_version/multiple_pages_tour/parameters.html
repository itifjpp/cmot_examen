Multiple pages tour (page 2)

This is the tables example page, which we use for the second part of a tour that runs on a different page where the tour started.

This tour starts automaticlly once the page gets loaded by using the parameter url trigger.

Once this tour is done we will redirect you to the last part of the tour.
